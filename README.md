🏥 Clinic & Pharmacy Management System

A simple and modern Clinic & Pharmacy Management Web App for managing stock, billing, patients, and reports.

---

🌐 Live Demo

👉 https://wahidzaman-prog.github.io/clinic-pharmacy-system/
